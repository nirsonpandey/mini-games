# Number Guessing Game Portfolio Website
# About the Project

This project is a personal portfolio website that showcases my skills, projects, and contact information while featuring an interactive Number Guessing Game built using HTML, CSS, and JavaScript. The website combines a modern portfolio with a fun browser game, allowing visitors to learn about me and interact with one of my projects in the same place.

# The Number Guessing Game generates a random number, and the player must guess the correct number within 10 attempts. After each guess, the game provides hints indicating whether the correct number is higher or lower, creating an engaging and interactive experience.

# How to Play
Open the portfolio website in your web browser.
Navigate to the Number Guessing Game section.
Enter a number in the input field.
Click the Guess button.
Read the hint provided by the game.
Continue guessing until:
You guess the correct number, or
You use all 10 attempts.
Click Restart to play again.
