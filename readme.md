# Number Guessing Game
# About the Project

This is a simple and interactive Number Guessing Game built using HTML, CSS, and JavaScript. The game generates a random number, and the player must guess the correct number within 10 attempts. After each guess, the game provides feedback to help the player determine whether the correct number is higher or lower than their guess.

# Features
Random number generation
Maximum of 10 guessing attempts
Hints after each guess (Too High / Too Low)
Win and Lose conditions
Input validation
Clean and responsive user interface
Option to restart the game
Technologies Used
# HTML
Creates the structure of the web page.
Includes input fields, buttons, and result display sections.
# CSS
Styles the web page and improves its visual appearance.
Makes the layout responsive and user-friendly.
# JavaScript
Generates a random number.
Handles user input and guess validation.
Compares guesses with the generated number.
Tracks the number of attempts.
Displays feedback and game results.
# How to Play
Open the game in your web browser.
Enter a number in the input field.
Click the Guess button.
Read the hint provided by the game.
Continue guessing until:
You guess the correct number, or
You use all 10 attempts.
Restart the game to play again.
